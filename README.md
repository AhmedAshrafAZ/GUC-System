## Project Installation

### With Back-end

1. Download the project

   ```
   git clone https://github.com/AhmedAshrafAZ/GUC-System.git
   ```

2. Enter the project directory

   ```
   cd GUC-System
   ```

3. Install the node modules

   ```
   npm i
   ```

4. Run the project

   ```
   node index.js
   ```
